# Sprint 1: Project Structure

## Goal

Understand and document the initial Spring Boot project structure.

## Tasks

- Run the application locally
- Understand the Maven project structure
- Understand the Spring Boot main class
- Understand the purpose of `pom.xml`
- Understand the purpose of `application.properties`
- Document the initial architecture
- Create the first Architecture Decision Record

## Acceptance Criteria

- The application starts successfully
- Maven tests run successfully
- The project structure is documented
- The Spring Boot framework decision is documented
- Changes are committed in a feature branch
